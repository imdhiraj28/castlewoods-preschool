# 🏰 Castlewood International Pre-School

A modern, responsive website for **Castlewood International Pre-School** — a trusted early childhood education institution in Indore, India, nurturing young learners since 2013.

> *"Learn with Pride"*

## ✨ Features

- **Fully Responsive** — Optimized for desktop, tablet, and mobile
- **Modern Design** — Navy blue & gold brand identity with smooth gradients and glassmorphism
- **Scroll Animations** — Elements fade in as they enter the viewport
- **Testimonial Carousel** — Auto-advancing reviews with manual controls
- **Admission Enquiry Form** — Client-side validated contact form
- **Smooth Navigation** — Sticky header with smooth-scroll anchor links
- **Counter Animations** — Animated statistics in the hero section
- **Mobile Menu** — Responsive hamburger navigation

## 🚀 Getting Started

### View Locally

Simply open `index.html` in your browser:

```
index.html
```

### Serve with Local Server

```bash
# Python
python -m http.server 8080

# Node.js
npx serve .
```

Then visit `http://localhost:8080`
